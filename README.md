# 🪴 LiClass(라이클래스)
Like(좋아하다) + Class(클래스 취미활동)
<br><br>
: 직접 만나 취향과 경험을 공유하고, 재밌게 배우는 것을 통해 평범한 일상을 특별하게 기록할 수 있는 다양한 오프라인 원데이 클래스가 활성화될 수 있는 플랫폼을 기획해보았습니다.
<br><br>

## 📍 서비스 소개 및 기획의도

코로나19를 지나며 현대인들의 취미/자기 계발 분야에 대한 관심은 지속적으로 높아지고 있지만<br> 
자신만의 관심사를 충족하는 적절한 취미활동에 대한 정보수집, 동기부여는 여전히 부족한 상황입니다.<br>
오프라인으로 원데이클래스, 취미공유활동 기획자의 입장에서도 자신의 프로그램들을 홍보하고 활성화할 기회가 부족한 상황이었습니다.<br>
이들을 연결하고 소통을 활성해줄 수 있는 온라인 플랫폼! 이라는 해답을 찾게 되어 이를 만들어보기로 하였습니다.

원데이 클래스 정보를 주제별, 지역별, 강사별로 제공하여 일반 사용자들에게 충분한 정보전달과 제공하고자 하였습니다.<br>
더불어 예약, 결제 시스템을 구축하여 사용자가 편리하고 빠른 이용을 즐길 수 있도록 하였습니다.<br>
시스템 관리자에게는 플랫폼의 사용자, 트래픽 정보를 다방면에서 통계분석하여 제공하여 효율적인 운영환경을 제공하고자 하였습니다.

## 📍 개발환경 및 기술
<img width="600" alt="image" src="https://github.com/user-attachments/assets/991cbdca-12c7-4bd0-9224-d456d3ee8f72"><br>

- OS : Windows 10
- DBMS : Oracle Database Express Edition (XE) Release 18.4.0.0.0 (18c)

- API : 포트원 결제/환불API,카카오 로그인/맵/카카오톡공유API,네이버 로그인/블로그공유API,SNS 공유 API(트위터,페이스북)
